分支的新建、切换、合并

Git 是怎么创建新分支的呢? 很简单，它只是为你创建了一个可以移动的新的指针。比如创建一个 testing 分支

git branch testing

切换分支

git checkout testing

另外, 创建与切换分支可合成一步

git checkout -b testing

添加新的提交

切换回master分支

git checkout master

合并testing分支到master

git merge testing

合并时，可能会有冲突，这时需要解决冲突，解决冲突用GUI比较为方便, 命令行可以使用git mergetool
